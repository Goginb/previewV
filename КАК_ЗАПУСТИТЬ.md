# Как запустить PreviewV

Полная инструкция по установке, разработке, сборке, горячим клавишам и проектам — в корневом файле **[README.md](README.md)** (он же отображается на главной странице репозитория на GitHub).

## Для обычных пользователей

Если вам нужно просто установить приложение, а не разрабатывать его:

- скачайте **`PreviewV Setup.exe`** из [GitHub Releases](https://github.com/Goginb/previewV/releases), по [прямой ссылке на установщик](https://github.com/Goginb/previewV/releases/latest/download/PreviewV%20Setup.exe) или возьмите его из папки **`release\`**
- запустите файл двойным кликом
- приложение установится для текущего пользователя без лишнего мастера установки

Если установка не нужна, можно использовать **`PreviewV Portable <версия>.exe`**.

## Быстрый старт (разработка)

### Вариант 1: батник

```bat
start.bat
```

`start.bat` сам проверит `Node.js`, установит зависимости через `npm ci`, если они ещё не установлены, и затем поднимет `npm run dev`.

### Вариант 2: вручную

```powershell
cd путь\к\PreviewV
npm ci
npm run dev
```

В терминале должна появиться строка с `Loading renderer from Vite dev server`; в окне справа снизу — метка **`DEV`**.

Если интерфейс «старый»:

```powershell
npm run clean
npm run dev
```

## Сборка Windows

**Установщик** (простой one-click install для обычных пользователей):

```powershell
npm run build:win
```

Файл вида **`release\PreviewV Setup.exe`**.

Или просто:

```bat
build-installer.bat
```

**Оба файла сразу для передачи другим людям**:

```bat
build-shareable.bat
```

**Только portable** (один exe, без установки и без ярлыков):

```powershell
npm run build:win:portable
```
