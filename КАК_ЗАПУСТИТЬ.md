# Как запустить PreviewV

Полная инструкция по установке, разработке, сборке, горячим клавишам и проектам — в корневом файле **[README.md](README.md)** (он же отображается на главной странице репозитория на GitHub).

## Быстрый старт (разработка)

```powershell
cd путь\к\PreviewV
npm install
npm run dev
```

В терминале должна появиться строка с `Loading renderer from Vite dev server`; в окне справа снизу — метка **`DEV`**.

Если интерфейс «старый»:

```powershell
npm run clean
npm run dev
```

## Сборка Windows

**Установщик** (мастер, выбор папки, ярлыки после установки):

```powershell
npm run build:win
```

Файл вида **`release\PreviewV Setup 0.2.0.exe`**.

**Только portable** (один exe, без установки и без ярлыков):

```powershell
npm run build:win:portable
```
